A minimalist glass-style widget set featuring a high-impact clock and system stats.

![Preview](preview.png?v=1.0)

## Installation
1. **[Download gall.widget.zip](https://github.com/msavox/Glass-Widgets-Uebersicht/raw/main/gall.widget.zip)**
2. Unzip the file into your Übersicht widgets folder.
3. For system stats (Fans & Temp) on Intel Macs, run:
   `sudo chmod +s /usr/bin/powermetrics`

## Included Widgets
- **Clock**: Large typography with date and time.
- **Calendar**: Minimalist monthly view.
- **CPU & RAM**: Live system monitoring.
- **Disk Usage**: Overview of your storage.
- **Network**: Real-time traffic stats.
- **Weather**: Forecast with glassmorphism icons.
- **Misc**: Extra system information (Fan speed, Temp).
