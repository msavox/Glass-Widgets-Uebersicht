# Glassmorphism Widgets for Übersicht - v1.1
# msavox 2026

A minimalist glass-style widget set featuring a high-impact clock, system stats, and interactive positioning.

## New in v1.1: Interactivity
- **Draggable Layout**: Click and drag any widget to reposition it anywhere on your desktop.
- **Lock System**: Each widget features a small lock icon (🔓/🔒). Lock a widget to freeze its position and prevent accidental drags.
- **Persistence**: Your custom layout is automatically saved and persists across reboots and Übersicht refreshes.

## Installation
1. **[Download gall.widget.zip](https://github.com/msavox/Glass-Widgets-Uebersicht/raw/main/gall.widget.zip)**
2. Unzip the file into your Übersicht widgets folder.
3. **Enable Interaction**: In Übersicht settings, ensure "Interaction" is enabled for these widgets to allow dragging and locking.
4. For system stats (Fans & Temp) on Intel Macs, run:
   `sudo chmod +s /usr/bin/powermetrics`

## Included Widgets
- **Clock**: Large typography with date and time.
- **Calendar**: Minimalist monthly view.
- **CPU & RAM**: Live system monitoring.
- **Top Processes** (New!): Real-time process monitoring (CPU Core%, System Load%, and RAM usage) inspired by `top/btop`.
- **Disk Usage**: Overview of your storage.
- **Network**: Real-time traffic stats with smoothing.
- **Weather**: Forecast with glassmorphism icons (Open-Meteo API).
- **Misc**: Extra system information (Fan speed, GPU, Temp).
